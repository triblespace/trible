# Trible CLI

> **This repository has moved.**
>
> The `trible` CLI now lives inside the
> [`triblespace-rs`](https://github.com/triblespace/triblespace-rs) workspace
> as a first-class member crate. Further development, issue tracking, and
> releases happen there.

## Where to go

- **Source:** [`triblespace-rs/trible`](https://github.com/triblespace/triblespace-rs/tree/main/trible)
- **Issues & PRs:** [triblespace-rs/issues](https://github.com/triblespace/triblespace-rs/issues)
- **Book (the distributed-sync chapter covers the CLI's `pile net` surface):**
  [triblespace book](https://docs.rs/triblespace/)

Published `trible` crate versions on crates.io continue to work for
users who install via `cargo install trible` — the underlying source
just lives in a different repo now.
